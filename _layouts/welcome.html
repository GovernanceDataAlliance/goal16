The SDG16 Data Initiative

“Promote peaceful and inclusive societies for sustainable development, provide access to justice for all and build effective, accountable and inclusive institutions at all levels”

The SDG16 Data Initiative is a collective project to gather and curate existing global data to track Sustainable Development Goal (SDG) 16. We draw on a multitude of data sources to track progress towards achieving SDG16.

This is a beta version of our website - please consider it work in progress!
